    DatabaseHelper

    This Class implements some useful database utilities. Many copied from TestDataFactory. It is pretty dangerious to use these in code, but sometimes you need to deactivate triggers to repair or migrate data. So these are available to call from anonymous apex... @version 2021-10-27 @todo update TestDataFactory to call this class, rather than having a duplicate implementation.
