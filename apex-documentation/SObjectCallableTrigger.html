    SObjectCallableTrigger

    This is mainly intended as a sample callable trigger class. In this case it is a trigger we can assign to any SObject to copy the standard field, OwnerId, to a custom field, Owner__c. This is useful if you want to follow the lookup onto the owner record in queries and such.

    If we have core trigger functionality we want to span across many SObject types, this the the class to implement it in.
    Signature
    public without sharing class SObjectCallableTrigger implements Callable
