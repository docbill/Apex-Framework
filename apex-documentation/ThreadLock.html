    ThreadLock

    This is a class for thread locking. The normal usage is: <pre> final String MY_THREAD_KEY = '<<<This is a unique value.>>>'; if(ThreadLock.lock(MY_THREAD_KEY)) { try { // do some work } finally { ThreadLock.unlock(MY_THREAD_KEY); } } </pre> @version 2020-09-04
